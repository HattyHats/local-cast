# Local-Cast: The Peer-to-Peer Operating System

Local-Cast is a futuristic, highly secure, browser-based operating system designed for instant, local file sharing and collaboration. It completely bypasses traditional cloud servers, meaning your files and data never leave your local network.

## Core Features

- **P2P WebRTC Engine**: Connect directly to other devices in the room by simply scanning a QR code. File transfers are handled device-to-device with no middleman server.
- **Virtual File System (VFS)**: A robust, persistent file system built on IndexedDB that allows you to drag-and-drop massive files and complex folder structures directly into the browser.
- **Granular Security**: Fully password-protect the entire server or lock down specific folders. The advanced security engine mathematically hides the contents of locked folders from unauthorized guests, ensuring bulletproof privacy.
- **Collaborative Toolkit**: Complete with an encrypted local chat room for instant messaging, and a built-in text editor to rapidly jot down and share notes across devices.
- **Immersive Media Viewer**: Seamlessly preview images, watch videos, and read text files natively within the browser without ever having to download them.
- **Burn Protocol**: A single, global "Burn Server" button that instantly nukes the file system and wipes all chat history across every connected device simultaneously.
- **Progressive Web App (PWA)**: Installable directly to your home screen or dock. It functions as a native application with robust offline caching capabilities.

*Local-Cast is the ultimate tool for secure, rapid, and visually stunning local collaboration.*
